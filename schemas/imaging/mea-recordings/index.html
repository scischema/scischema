Multi-Electrode Array (MEA) Recordings

A SciSchema.org Process Schema

Domain: Imaging & Measurement
Schema: MEA Recordings Process Schema

Multi-electrode array recordings are used to measure the electrical activity of cells using a grid of microelectrodes.

Properties from Multi-Electrode Array (MEA) Recordings

The table below lists the top-level properties of this master schema. Each property may contain more detailed nested fields in the JSON Schema representation.

Property Expected Type Description Required
sampleType Object Biological preparation characteristics, culture conditions, and maturation status Yes
electrode Object Electrode physical, electrical, and configuration characteristics including specialized poration methods Yes
acquisition Object Data acquisition system configuration with explicit distinction between hardware and software processing stages Yes
environment Object Environmental control parameters for physiological maintenance No
stimulation Object Electrical, optical, or pharmacological stimulation parameters (optional for passive recordings) No
dataProcessing Object Data processing parameters, software, and computational methods used for analysis No
outputMetrics Array Target metrics to be extracted from recordings. Specifies analysis focus during acquisition setup. Field potential duration, corrected metrics, and action potential-related metrics (derived from field potential measurements) are particularly relevant for cardiac applications. Short-term variability (STV) is critical for assessing arrhythmia risk. Conduction velocity is essential for estimating sodium current effects. No
