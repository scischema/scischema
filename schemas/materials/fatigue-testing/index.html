Fatigue Testing of Metallic Materials

A SciSchema.org Process Schema

Domain: Materials & Chemistry
Schema: Fatigue Testing of Metallic Materials

Fatigue testing is a destructive mechanical testing process in which a metallic specimen is subjected to repeated cyclic loading until failure occurs.

Properties from Fatigue Testing of Metallic Materials

The table below lists the top-level properties of this master schema. Each property may contain more detailed nested fields in the JSON Schema representation.

Property Expected Type Description Required
material Object Details of the metallic material being tested. Yes
specimenGeometry Object Physical dimensions and shape of the test specimen, conforming to standards such as ASTM E466 or ISO 1099. Yes
loadingType String Mode of mechanical loading applied during the test. Yes
loadingParameters Object Parameters defining the loading conditions applied during testing. One of stressAmplitude or strainAmplitude applies depending on the test control type. Yes
testEnvironment Object Environmental conditions during the test that may influence crack initiation and propagation. Yes
equipment Object Details of the testing equipment and standards used. Yes
outputs Object Results and outputs from the fatigue test. Yes
