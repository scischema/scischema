Electrical Cell Stimulation

A SciSchema.org Process Schema

Domain: Biology & Biotechnology
Schema: Electrical Cell Stimulation Process Schema (own)

Electrical cell stimulation is a technique in which controlled electric currents or fields are applied to cells to influence their behavior and function.

Properties from Electrical Cell Stimulation

The table below lists the top-level properties of this master schema. Each property may contain more detailed nested fields in the JSON Schema representation.

Property Expected Type Description Required
experiment_metadata Object Metadata about the experiment, including general information and timing. No
cell_culture Object Parameters related to cell preparation and culture conditions. Yes
electrical_stimulation Object Parameters defining the electrical stimulation configuration. Yes
experimental_procedure Array Step-by-step description of the experimental workflow. Yes
experimental_analysis Object Methods and materials used for post-stimulation evaluation. Yes
